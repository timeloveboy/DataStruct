## 树的运算

+ 先序遍历：根左右：有左就先做，没左再右

``` java
         1 
       /   \
      2     5
     /     / \
    3     6   7
     \
      4 
    
```
+ 中序遍历:左根右：有左右，就先左右，再根，

``` java
         4 
       /   \
      3     6
     /     / \
    1     5   7
     \
      2 
```
+ 后序遍历:左右根

``` java
         7 
       /   \
      3     4
     /     / \
    2     5   6
     \
      1 
```
+ 层次遍历： 

``` java
         1 
       /   \
      2     3
     /     / \
    4     5   6
     \
      7 
```
